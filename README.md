### 기획

  기획안 작성 시 '추후지정'이란 없다. 기획안은 목표는 나의 계획을 자신 있게 관철시키는 것에 있다. 
  
  따라서 자신없는 표현은 금뭉, 도려내야 한다. 
  
  예) '코딩하는 유튜버' 기획안 작성 건

### 기획연재 - 팀 연재

  - 취재 지표가 내 스스로 확정되 있어야 할 것
